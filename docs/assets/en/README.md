# 영어판 스크린샷

영어 README(`README.en.md`)용 스크린샷입니다. 한국어 스크린샷(`../`)의 UI 텍스트를
영어로 옮긴 것으로, **실제 확장을 영어로 띄워 찍은 실물이 아닙니다.**

## 실물과 다른 부분 (재촬영 시 반영할 것)

- **언어 설정 카드가 없습니다.** 원본이 언어 선택 기능(v0.3.0) 추가 전 스크린샷입니다.
- **문구가 확장과 미세하게 다릅니다.**
  - `This increases the session file size` → 실제는 `Turn off for lighter session files`
  - `Changes are saved automatically. No save button needed.` → 실제는 `Changes save automatically. There's no save button.`
- **저장 경로가 옛 기본값입니다.** `~/Desktop/Archive/web-chats` → 현재 기본값은 `~/web-chats` (v0.2.0에서 변경).

## 교체 방법

확장 설정에서 언어를 English로 바꾼 뒤 팝업·설정·도움말 화면을 직접 촬영해 이 폴더의
파일을 덮어쓰면 됩니다. 그때 위 목록도 함께 지워주세요.
